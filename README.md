# Breast Cancer Diagnosis Prediction (Applied Statistics + ML)

This project analyzes breast cancer tumor features to classify diagnosis (Benign vs Malignant).  
It includes data cleaning, exploratory analysis, correlation assessment, Wilcoxon (Mann–Whitney) testing, and an SVM classifier with ROC/AUC evaluation.

## Files
- `breast_cancer_project.Rmd` — full analysis
- `data/breast_cancer.csv` — dataset (if uploaded)
- `images/` — visualizations used in results

## How to run
1. Open `breast_cancer_project.Rmd` in RStudio
2. Install packages (first run only)
3. Click **Knit** to generate the HTML report
